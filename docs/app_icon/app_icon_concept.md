# App アイコン デザインコンセプト

## 一言コンセプト

**「中身が知識グラフになっている1枚のノート」** — ノート(カード)の内部に、意味の
近さで結ばれたノード＋リンクのグラフを描くことで、「ノート」と「意味/ベクトル検索」という
2つの概念を1つのシルエットに統合する。SemanticNotes の本質(ノート＝意味のネットワーク)を
そのまま図像化したもの。

## デザイン要素と対応

| 要素 | 表現 | コンセプトとの対応 |
|---|---|---|
| ノート/カード | 角丸の縦長カード(ドキュメントと一目で分かる形) | 「ノート」 |
| ノード＋リンク | カード上の5ノード＋放射状リンク。中央=検索クエリ、周囲=意味的近傍 | 「意味検索」「ベクトル検索」 |
| 光・星 | 中央ノードのシアン発光(グロー)＋4点スター(きらめき) | 「AI/オンデバイスAI」 |

- **波形は不採用**: 小サイズで潰れて視認性を下げるため。星＋グローの方が「知性/AI」を端的に伝えられると判断。
- **文字なし**: 小サイズでの可読性と普遍性のため(要件6)。
- **他社ロゴなし**: Apple / OpenAI / Claude 等のロゴ・記号は一切使用しない(要件7)。

## レイアウト方針

- キャンバスは 1024×1024。**フルブリード**(自前の角丸なし・不透明背景)。iOS がホーム画面で角丸マスクを
  自動適用するため、原本は四角のまま作る。
- カードを画面中央にほぼ垂直で配置(傾けない)。小サイズでもカードのシルエットが最優先で読めるようにする。
- グラフはカード内側に余白を取って収める。中央ノードを最大・発光させ、視線を集める。

## 反映手順(承認後)

`Assets.xcassets/AppIcon.appiconset` に単一サイズ(1024)の不透明 PNG を配置済み(`AppIcon-1024.png`)。
App Store 検証は alpha チャンネルを弾くため、alpha を除去してある。

alpha 除去の方法(qlmanage 出力は alpha 付きのため):

```bash
python -c "from PIL import Image; Image.open('app_icon_1024.png').convert('RGB').save('app_icon_1024.png')"
```

`convert('RGB')` は alpha チャンネルを破棄する。背景がフルブリードで全画素が不透明なため、見た目は不変。
